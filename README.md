# Brain OS Marketplace

Plugin marketplace for [Brain OS](https://github.com/sonthanh/brain-os-plugin) — a Claude Code plugin for managing a second brain with Obsidian.

## Install

```bash
/plugin marketplace add sonthanh/brain-os-marketplace
/plugin install brain-os@brain-os-marketplace
```

## Update

```bash
/plugin update brain-os
```
